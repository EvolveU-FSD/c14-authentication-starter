Cookie Based Authentication

0 - check if you already have a cookie from before (/cookie/user)

1 - Ask the server for a session cookie by providing (/cookie/login)

Username
Password

2 - Check the browser console to see the cookie in use (/cookie/balance)

3 - Log out by telling the server you are done

The server may reset the cookie to be blank value, or just forget the session on the server side so the next time that cookie is presented it will be ignored
